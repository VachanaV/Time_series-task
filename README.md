Task 3 – Time Series Analysis

Dataset: Airline Passengers Dataset (1949–1960)

Techniques Used
• Time Series Decomposition
• Moving Averages
• ARIMA Forecasting

Observations

Trend:
Passenger numbers increase steadily from 1949 to 1960.

Seasonality:
Passenger traffic peaks around mid-year every year.

Forecast:
ARIMA model predicts future passenger growth with low error (RMSE).
